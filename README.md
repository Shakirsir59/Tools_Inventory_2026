# 🛠️ North Region Unified Master Tracker

Telecom Infrastructure Assets & Rigger Safety Tool Engine. Yeh system North Region ke field personnel (riggers) ke paas available telecom hardware aur safety gears ko track aur manage karne ke liye banaya gaya hai.

## 🚀 Live Dashboard
Aap is tracker ka live interactive dashboard yahan dekh sakte hain:
👉 **[https://shakirsir59.github.io/North-Tools-Inventory/](https://shakirsir59.github.io/North-Tools-Inventory/)**

---

## 📊 Project Features
* **Dynamic Pie Chart:** Telecom models ke deployment ka real-time percentage autographed breakdown.
* **Live Search Filter:** Employee name ya device details se instantly records filter karne ki suvidha.
* **Color-Coded Alert System:** Kisi bhi staff ke pass tool shortage (`Need`), `Damage`, ya extra items hone par instant red alert border highlight.
* **Fully Responsive:** Mobile aur desktop dono par smoothly chalta hai (Powered by Tailwind CSS).
